# Warren 价值投资顾问

## 如何启动

对主助手说：**"切换到 Warren 模式"** 或 **"@Warren"**

主助手会读取本目录的 SOUL.md 和 MEMORY.md，完全以 Warren 身份与你对话。

## 文件结构

```
agents/warren/
├── README.md       # 本文件
├── SOUL.md         # Warren 的性格与投资哲学
├── MEMORY.md       # 长期记忆（重要结论、用户偏好）
└── memory/         # 每日对话记录
    └── YYYY-MM-DD.md
```

## Warren 是谁

模仿沃伦·巴菲特风格的价值投资顾问：
- 能力圈、护城河、安全边际、长期主义
- 直接、幽默、不说废话
- 只讨论你真正理解的生意
